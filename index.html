首次使用需要连接云数据库，按以下步骤操作一次，以后直接打开就能用。
1
打开 supabase.com，注册免费账号（用 GitHub 或邮箱）
2
点击 New Project，填项目名（如 schedule），记住你设置的数据库密码
3
进入项目后，点左侧 SQL Editor，粘贴下方 SQL 并运行：

create table teachers (id serial primary key, name text, color text, created_at timestamptz default now()); create table lessons (id serial primary key, teacher_id int references teachers(id), date text, slot int, name text, color text, note text, created_at timestamptz default now()); alter table teachers enable row level security; alter table lessons enable row level security; create policy "allow all" on teachers for all using (true) with check (true); create policy "allow all" on lessons for all using (true) with check (true);
4
点左侧 Settings → API，复制 Project URL 和 anon public key，填入下方
